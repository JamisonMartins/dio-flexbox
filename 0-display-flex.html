Usando Display: flex

Este documento HTML há uma única <section> de classe flex que possui a propriedade display com valor flex que alinhará seus itens em linha quando o a tela do dispotivo ter pelo menos 500 pixels de largura, mesmo que por padrão seu compartamento seja a de ocupar todo espaço na horizontal, também conhecidos como item com display block.

Dentro desta <section> há três divs que ficam uma ao lado da outra por conta da propriedade css display da <section class="flex">. Abaixo o código CSS da classe flex .

.flex {
   margin: 0 auto;
   max-width: 900px;
   padding: 10px;
   display: flex;
   background-color: rgba(255, 182, 193, 0.12);
}

Segunda div

Este parágrafo pertence a segunda div que está dentro da section citada anteriormente. As divs possuem a classe item e possui as seguintes propriedades CSS

.item {    border: 1px solid blueviolet;
   margin: 5px;
   padding: 5px 10px;
}

Os conteúdos das tags <p> das divs classe item possuem alinhamento justificado.

.item p {    text-align: justify; }
